    Crash Tracking System

    This system is based on the application of IoT (Internet of Things) which was built to detect motorcycle crash accidents using a vibration sensor and a gyroscope tilt sensor. Information messages are sent over the GPRS (General Packet Radio Service) network and a GPS (Global Positioning System) module is used to determine the coordinates of the location for tracking purpose. In this system uses an interface of a web application and use the MQTT (Message Queuing Telemetry Transport) protocol as a data transfer procedure between the web and the device. This system also utilizes the telegram application in the form of a telegram chat bot which is used as a recipient of information notifications to mobile phones (smartphones) so that they can track the locations.
